Reproduzindo audios

Outra maneira

Para voce colocar audio, basta usar a tag; audio e colocar o controls dentro da tag

Como colocar vídeos

para adicionar vídeo é só add a tag video e dentro da tag colocar controls para mostrar os controles
caso o vídeo fique muito grande na tela, usa width = "500" por exemplo
é interessante saber que tem como colocar varios tipos de videos e pedir para o html ir testando
para evitar de uma navegador não ser compativel com o tipo do video

Para usar uma capa no vídeo, usar a tag poster dentro da tag video

Videos de hospedagem local é caro, existe uma forma melhor

Incorporação de videos externos

É só ir no youtube apertar em compartilhar e depois em incorporar, contrl+c e control+v
